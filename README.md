## Vue2 Webpack2 Boilerplate

### Installation
```
yarn
```
### Development

#### Running the Dev Server

```
Save `.env.defaults` file as `.env`
```

##### Run the webpack dev server
```
`yarn start:dev`
```
##### Run the App in your browser
```
Go to http://localhost:8080/
```

### Technology

- Yarn
- Webpack 2
- Vue2
- Vue Router
- SASS
